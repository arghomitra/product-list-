# **App Name**: ProList

## Core Features:

- Item List Display: Display items with quantity input fields for easy adjustment.
- Dynamic List Management: Use AI to find similar or suggested items based on those in the list, and offer suggestions for inclusion as a tool.
- Admin Authentication: Secure admin interface with password protection to add or modify items in the list.
- PDF Generation: Generate a professional PDF document containing the list of items and their quantities, with extra space for a custom note.
- Data Persistence with Cookies: Save the selected items, quantity and notes using browser cookies to maintain state across sessions.
- Download and Share: Enable users to easily download the PDF and send it via WhatsApp or other messaging platforms.
- Print Functionality: Provide a print button for direct printing of the item list, to streamline the usage of this MVP app in real scenarios.

## Style Guidelines:

- Primary color: Navy blue (#2E3A87) for professionalism and reliability.
- Background color: Light gray (#F0F0F0) for a clean, modern look.
- Accent color: Teal (#008080) to provide subtle contrast and call attention to buttons.
- Body and headline font: 'PT Sans', a humanist sans-serif with a blend of modern design and a touch of warmth, for readability and a professional look.
- Code font: 'Source Code Pro' for displaying admin passwords.
- Simple, clear icons for actions like download, print, and share.
- Clean, structured layout with items listed clearly, quantity inputs easily accessible, and clear call to action buttons.
